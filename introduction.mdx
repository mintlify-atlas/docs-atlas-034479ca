## What is MeshLib?

MeshLib is an open-source library that provides advanced algorithms for 3D data processing. It helps developers and engineers achieve precise results while delivering significant resource savings. Whether you are working on 3D printing, scanning, inspection, automation, robotics, scientific visualization, or medical devices, MeshLib is built to meet your needs through its powerful algorithms and broad language support.

MeshLib is written in C++ and provides bindings for Python, C, and C#. Its foundation is a robust half-edge data structure that ensures manifold compliance for precise, reliable mesh representation. The library supports all essential 3D formats — point clouds, meshes, and volumes — as generated by modern sensors and processing pipelines.

## Use cases

**3D printing** — Prepare meshes for additive manufacturing by repairing defects, applying precise offsets for wall thickness, and simplifying geometry to reduce file size without losing critical detail.

**3D scanning and inspection** — Align captured scans to reference models using ICP registration, detect surface deviations, and convert raw point clouds to watertight meshes for downstream analysis.

**Industrial inspection** — Automate quality control pipelines in C++ or Python to detect collisions, measure surface offsets, and validate tolerance compliance across large part batches.

**Robotics** — Process environment scans and CAD geometry at runtime for path planning, obstacle avoidance, and grasp planning, with performance suitable for embedded and real-time systems.

**Medical devices** — Handle DICOM volumes and patient-specific mesh data with the precision required for surgical planning, prosthetic design, and regulatory-grade 3D analysis.
