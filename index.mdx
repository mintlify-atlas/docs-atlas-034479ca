## Get started

<Steps>
  <Step title="Install MeshLib">
    For Python users, install with a single command:
    ```bash
    pip install meshlib
    ```
    For C++, C, or C#, see the [Installation guide](/installation).
  </Step>
  <Step title="Load a mesh">
    ```python
    import meshlib.mrmeshpy as mrmeshpy

    mesh = mrmeshpy.loadMesh("model.stl")
    print(f"Loaded mesh with {mesh.topology.numValidFaces()} faces")
    ```
  </Step>
  <Step title="Process your mesh">
    ```python
    # Decimate (simplify) the mesh
    settings = mrmeshpy.DecimateSettings()
    settings.maxError = 0.05
    mrmeshpy.decimateMesh(mesh, settings)
    ```
  </Step>
  <Step title="Save the result">
    ```python
    mrmeshpy.saveMesh(mesh, "output.stl")
    ```
  </Step>
</Steps>
