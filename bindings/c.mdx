title: "C API"
description: "Use MeshLib from C via the MeshLibC interface"
---

MeshLib provides a C-compatible API through the `MRCMesh` headers. All types are prefixed with `MR_` and all functions follow a `MR_TypeName_methodName` naming convention, making the API usable from any C89/C99 toolchain or FFI.

## Headers and linking

```c
#include <MRCMesh/MRMesh.h>
#include <MRCMesh/MRMeshLoad.h>
#include <MRCMesh/MRMeshSave.h>
#include <MRCMesh/MRMeshBoolean.h>
#include <MRCMesh/MRAffineXf.h>
#include <MRCMesh/MRMakeSphereMesh.h>
#include <MRCMesh/MRVector3.h>
#include <MRCMisc/expected_MR_Mesh_std_string.h>
#include <MRCMisc/expected_void_std_string.h>
#include <MRCMisc/std_string.h>
```

Link against `MeshLibC` in your build system alongside the core `MRMesh` library.

## Memory ownership

The C API uses explicit heap allocation. Every object created by a `MR_*_Construct*` call or returned by a factory function must be freed with its matching `MR_*_Destroy` function. Expected wrappers (`MR_expected_*`) must also be destroyed after use, even on the error path.

<Warning>
  Forgetting to call `MR_Mesh_Destroy`, `MR_BooleanResult_Destroy`, or `MR_expected_*_Destroy` will leak memory. Always use `goto`-based or equivalent cleanup patterns as shown in the examples below.
</Warning>

## Loading and saving meshes

```c mesh_load_save.c
#include <MRCMesh/MRMesh.h>
#include <MRCMesh/MRMeshLoad.h>
#include <MRCMesh/MRMeshSave.h>
#include <MRCMesh/MRMeshTopology.h>
#include <MRCMesh/MRString.h>
#include <MRCMisc/expected_MR_Mesh_std_string.h>
#include <MRCMisc/expected_void_std_string.h>
#include <MRCMisc/std_string.h>

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main( void )
{
    int rc = EXIT_FAILURE;

    // Load mesh.
    MR_expected_MR_Mesh_std_string* meshEx = MR_MeshLoad_fromAnySupportedFormat_2( "mesh.stl", NULL, NULL );
    MR_Mesh* mesh = MR_expected_MR_Mesh_std_string_value_mut( meshEx );

    if ( !mesh )
    {
        fprintf( stderr, "Failed to load mesh: %s\n", MR_std_string_data( MR_expected_MR_Mesh_std_string_error( meshEx ) ) );
        goto fail_load;
    }

    // Save mesh.
    MR_expected_void_std_string* saveEx = MR_MeshSave_toAnySupportedFormat_3( mesh, "mesh.ply", NULL, NULL);
    if ( MR_expected_void_std_string_error( saveEx ) )
    {
        fprintf( stderr, "Failed to save mesh: %s\n", MR_std_string_data( MR_expected_void_std_string_error( saveEx ) ) );
        goto fail_save;
    }

    rc = EXIT_SUCCESS;
fail_save:
    MR_expected_void_std_string_Destroy( saveEx );
fail_load:
    MR_expected_MR_Mesh_std_string_Destroy( meshEx );
    return rc;
}
```

## Boolean operations

```c mesh_boolean.c
#include <MRCMesh/MRAffineXf.h>
#include <MRCMesh/MRMakeSphereMesh.h>
#include <MRCMesh/MRMesh.h>
#include <MRCMesh/MRMeshBoolean.h>
#include <MRCMesh/MRMeshSave.h>
#include <MRCMesh/MRVector3.h>
#include <MRCMisc/expected_void_std_string.h>
#include <MRCMisc/std_string.h>

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main( void )
{
    int rc = EXIT_FAILURE;

    // First, create a unit sphere.
    float radius = 1.f; // Set radius for the test
    int horizontalResolution = 64; // Increase horizontal resolution
    int verticalResolution = 64; // Increase vertical resolution

    MR_Mesh* sphere1 = MR_makeUVSphere( &radius, &horizontalResolution, &verticalResolution );

    // Create a copy of this sphere and offset it.
    MR_Mesh* sphere2 = MR_Mesh_ConstructFromAnother( MR_PassBy_Copy, sphere1 );
    MR_Vector3f xfTranslation = {.x = 0.7f};
    MR_AffineXf3f xf = MR_AffineXf3f_translation( &xfTranslation );
    MR_Mesh_transform( sphere2, &xf, NULL );

    // Perform the boolean operation.
    MR_BooleanResult* result = MR_boolean_4_const_MR_Mesh_ref( sphere1, sphere2, MR_BooleanOperation_Intersection, NULL );
    if ( !MR_BooleanResult_valid( result ) )
    {
        fprintf( stderr, "Failed to perform boolean: %s\n", MR_std_string_data( MR_BooleanResult_Get_errorString( result ) ) );
        goto fail;
    }

    // Save result to an STL file.
    MR_expected_void_std_string* saveEx = MR_MeshSave_toAnySupportedFormat_3( MR_BooleanResult_Get_mesh( result ), "out_boolean.stl", NULL, NULL);
    if ( MR_expected_void_std_string_error( saveEx ) )
    {
        fprintf( stderr, "Failed to save mesh: %s\n", MR_std_string_data( MR_expected_void_std_string_error( saveEx ) ) );
        goto fail;
    }

    rc = EXIT_SUCCESS;
fail:
    MR_BooleanResult_Destroy( result );
    MR_Mesh_Destroy( sphere2 );
    MR_Mesh_Destroy( sphere1 );
    return rc;
}
```

## Key conventions

| Pattern | Example |
|---|---|
| Object creation | `MR_makeUVSphere(...)`, `MR_Mesh_ConstructFromAnother(...)` |
| Object destruction | `MR_Mesh_Destroy(ptr)`, `MR_BooleanResult_Destroy(ptr)` |
| Field access | `MR_BooleanResult_Get_mesh(result)`, `MR_BooleanResult_Get_errorString(result)` |
| Error string data | `MR_std_string_data(str_ptr)` |
| Expected value | `MR_expected_MR_Mesh_std_string_value_mut(ex)` — returns `NULL` on error |
| Expected error | `MR_expected_MR_Mesh_std_string_error(ex)` — returns `NULL` on success |

<Note>
  All `MR_expected_*` objects must be destroyed regardless of whether they hold a value or an error. Destroy them after you have finished reading both the value pointer and any error string.
</Note>
