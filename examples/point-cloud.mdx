MeshLib can load, process, and convert point clouds to meshes. The examples below cover the two main triangulation approaches and signed distance computation between meshes.

## Load and triangulate a point cloud

`triangulatePointCloud()` builds a surface mesh from an unstructured point cloud using a Poisson-style reconstruction.

```python points_to_mesh.py
from meshlib import mrmeshpy as mm
from pathlib import Path

# Load point cloud from file
pc = mm.loadPoints("Points.ply")

# Triangulate point cloud into a mesh
nefertiti_mesh = mm.triangulatePointCloud(pc)

# Save result
mm.saveMesh(nefertiti_mesh, "Mesh.ctm")
```

<Note>
  `triangulatePointCloud()` works best when the point cloud has consistent normals. If your cloud lacks normals, compute them first with `computeNormals()` before triangulating.
</Note>

## Fusion: smooth surface reconstruction

`pointsToMeshFusion()` reconstructs a smooth surface by fitting a signed distance field to the input points. It is more robust than direct triangulation for noisy scans.

```python fusion.py
from meshlib import mrmeshpy as mm

# Load point cloud
points = mm.loadPoints("Points.ply")

# Configure fusion parameters
params = mm.PointsToMeshParameters()
params.voxelSize = points.computeBoundingBox().diagonal() * 1e-2
params.sigma = max(params.voxelSize, mm.findAvgPointsRadius(points, 50))
params.minWeight = 1

# Reconstruct mesh via fusion
mesh = mm.pointsToMeshFusion(points, params)

# Save result
mm.saveMesh(mesh, "Mesh.ctm")
```

`sigma` controls the smoothing radius. Setting it to at least the average point spacing (returned by `findAvgPointsRadius`) prevents gaps in the reconstruction. `minWeight` discards voxels with fewer than this many contributing points, which removes outlier noise from the result.

## Signed distance between two meshes

`findSignedDistance()` computes the minimum signed distance between the surfaces of two meshes. The result is positive when the meshes do not penetrate each other and negative when they overlap.

```python signed_distance.py
from meshlib import mrmeshpy

# Load two meshes
mesh1 = mrmeshpy.loadMesh("mesh1.ctm")
mesh2 = mrmeshpy.loadMesh("mesh2.ctm")

# Compute signed distance
a = mrmeshpy.findSignedDistance(mesh1, mesh2)
print(f"Signed distance between meshes is {a.signedDist}")
```

## Collision detection

For detecting and inspecting collisions between two meshes, use `findCollidingTriangles()` or `findCollidingTriangleBitsets()`.

```python collision.py
from meshlib import mrmeshpy as mm

meshA = mm.makeUVSphere()  # make mesh A
meshB = mm.makeUVSphere()  # make mesh B
meshB.transform(mm.AffineXf3f.translation(mm.Vector3f(0.1, 0.1, 0.1)))  # shift for demonstration

# Find each pair of colliding faces
collidingFacePairs = mm.findCollidingTriangles(meshA, meshB)
for fp in collidingFacePairs:
    print(fp.aFace, fp.bFace)

# Find bitsets of colliding faces
collidingFaceBitSetA, collidingFaceBitSetB = mm.findCollidingTriangleBitsets(meshA, meshB)
print(collidingFaceBitSetA.count())  # number of colliding faces from mesh A
print(collidingFaceBitSetB.count())  # number of colliding faces from mesh B

# Fast boolean check: do the meshes collide at all?
isColliding = not mm.findCollidingTriangles(meshA, meshB, firstIntersectionOnly=True).empty()
print(isColliding)
```

<Tip>
  Pass `firstIntersectionOnly=True` to `findCollidingTriangles()` for a fast boolean yes/no collision check. This exits as soon as the first intersecting pair is found and avoids building the full list.
</Tip>
