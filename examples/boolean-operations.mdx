Boolean operations combine two meshes into one by computing their union, intersection, or difference. All operations go through the `boolean()` function, which returns a `BooleanResult` you must validate before using.

<Note>
  See the [boolean operations guide](/guides/mesh-boolean) for a full explanation of parameters, the `BooleanOperation` enum, and advanced options.
</Note>

## Union of two spheres

Combines both meshes into a single outer shell, discarding interior geometry.

<CodeGroup>

```python union.py
import meshlib.mrmeshpy as mrmeshpy

# Create first sphere with radius of 1 unit
sphere1 = mrmeshpy.makeUVSphere(1.0, 64, 64)

# Create second sphere by cloning the first and moving it in X direction
sphere2 = mrmeshpy.copyMesh(sphere1)
xf = mrmeshpy.AffineXf3f.translation(mrmeshpy.Vector3f(0.7, 0.0, 0.0))
sphere2.transform(xf)

# Perform union
result = mrmeshpy.boolean(sphere1, sphere2, mrmeshpy.BooleanOperation.Union)
if not result.valid():
    print(result.errorString)
else:
    mrmeshpy.saveMesh(result.mesh, "out_union.stl")
```

```cpp union.cpp
#include <MRMesh/MRMesh.h>
#include <MRMesh/MRMeshBoolean.h>
#include <MRMesh/MRMeshSave.h>
#include <MRMesh/MRUVSphere.h>
#include <iostream>

int main()
{
    // Create first sphere with radius of 1 unit
    MR::Mesh sphere1 = MR::makeUVSphere( 1.0f, 64, 64 );

    // Create second sphere by cloning the first and moving it in X direction
    MR::Mesh sphere2 = sphere1;
    MR::AffineXf3f xf = MR::AffineXf3f::translation( MR::Vector3f( 0.7f, 0.0f, 0.0f ) );
    sphere2.transform( xf );

    // Perform union
    MR::BooleanResult result = MR::boolean( sphere1, sphere2, MR::BooleanOperation::Union );
    if ( !result.valid() )
        std::cerr << result.errorString << std::endl;

    MR::Mesh resultMesh = *result;

    if ( auto saveRes = MR::MeshSave::toAnySupportedFormat( resultMesh, "out_union.stl" ); !saveRes )
    {
        std::cerr << saveRes.error() << std::endl;
        return 1;
    }

    return 0;
}
```

</CodeGroup>

## Difference of two spheres

Subtracts the volume of `sphere2` from `sphere1` (A − B).

<CodeGroup>

```python difference.py
import meshlib.mrmeshpy as mrmeshpy

# Create first sphere with radius of 1 unit
sphere1 = mrmeshpy.makeUVSphere(1.0, 64, 64)

# Create second sphere by cloning the first and moving it in X direction
sphere2 = mrmeshpy.copyMesh(sphere1)
xf = mrmeshpy.AffineXf3f.translation(mrmeshpy.Vector3f(0.7, 0.0, 0.0))
sphere2.transform(xf)

# Perform difference (sphere1 minus sphere2)
result = mrmeshpy.boolean(sphere1, sphere2, mrmeshpy.BooleanOperation.DifferenceAB)
if not result.valid():
    print(result.errorString)
else:
    mrmeshpy.saveMesh(result.mesh, "out_difference.stl")
```

## Error handling pattern

`boolean()` never throws. It returns a `BooleanResult` whose `valid()` method returns `false` on failure. Always check before accessing `result.mesh`.

```python
result = mrmeshpy.boolean(mesh_a, mesh_b, mrmeshpy.BooleanOperation.Union)
if not result.valid():
    print("Boolean failed:", result.errorString)
    # Optionally inspect which faces caused the problem:
    # result.meshABadContourFaces, result.meshBBadContourFaces
else:
    mrmeshpy.saveMesh(result.mesh, "output.stl")
```

<Warning>
  Input meshes must be free of self-intersections in the zone where they overlap. If either mesh has open boundaries in the intersecting region, operations like `Union`, `Intersection`, and `DifferenceAB` may fail. Repair the mesh first or use `OutsideA` / `OutsideB` for open meshes.
</Warning>
