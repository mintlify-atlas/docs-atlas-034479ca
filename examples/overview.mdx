These pages contain complete, runnable code examples for the most common MeshLib operations. Each example is taken directly from the MeshLib source repository and shown in Python, with C++ equivalents where available.

<Note>
  All Python examples require `meshlib` to be installed. Install it with:

  ```bash
  pip install meshlib
  ```
</Note>
